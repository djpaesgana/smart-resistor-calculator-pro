# Resistor Calc PRO

Resistor Calc PRO is an Android application designed to calculate resistor values. It uses a WebView to render a user-friendly interface powered by HTML/JS.

## Features
- Resistor color code calculation.
- WebView-based UI for cross-platform feel.

## Getting Started
To get a local copy up and running, follow these simple steps:
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the app on an emulator or physical device.
